# NEO Updater

Fetches potentially hazardous asteroid (PHA) data from NASA’s NeoWs API and writes a snapshot to `data/hazardous_neos/latest.json`.  
Runs locally and automatically 4×/day via GitHub Actions.  

---

## Credits
- Data: NASA NeoWs API
